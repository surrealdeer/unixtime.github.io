What is a Unix Timestamp?

Computers Emerge

As computers became useful utilities, a need to track time was necessary. Computers interacting with one another needed a way to share a time reference to one another and databases.

Efficient Time Tracking

Conventional dates contain various characters that need to be converted to a useful format each time they are used. By storing timestamps as an integer it is much easier for a computer to store and work for comparison using basic math.

January 1, 1970 - 12:00am UTC

The Unix Epoch is the starting point created as a universal reference. All Unix Timestamps are an integer that simply represent the number of seconds since that time (currently seconds since the Unix Epoch).
